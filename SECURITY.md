# 安全政策

## 支持的版本

目前仅最新版本会收到安全更新。

| 版本 | 是否支持 |
| ---- | -------- |
| latest | ✅ |
| older versions | ❌ |

## 报告安全漏洞

如果你发现了安全漏洞，请不要公开提交 Issue。

请通过以下邮箱联系我们：

kl@kldhsh.top

报告时请尽量包含以下信息：

- 漏洞描述
- 复现步骤
- 受影响的版本或提交
- 可能造成的影响
- 如果有修复建议，也可以一并提供

我们会尽量在 7 天内回复。

如果漏洞被确认，我们会尽快修复并发布安全更新。

感谢你帮助我们提升项目安全性。
